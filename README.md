# 🚦 Traffic Light Controller

A modern and interactive **Traffic Light Controller** web application that simulates real-world traffic signal management. The project provides automatic light sequencing, manual control options, pedestrian crossing support, emergency vehicle priority, and a responsive dashboard for monitoring traffic operations.

## 🌐 Live Demo

🔗 **Live Project:** https://gentle-maroon-p6drfj7f.edgeone.dev/

---

# 📖 Overview

The Traffic Light Controller is designed to demonstrate how smart traffic signals operate at road intersections. It offers both **automatic** and **manual** traffic control modes while providing a realistic user interface for educational purposes and smart city demonstrations.

The application focuses on improving traffic flow, reducing congestion, and simulating emergency situations.

# ⚙️ How It Works

1. Open the application.
2. Start the traffic controller.
3. Signals automatically rotate:
   - 🔴 Red
   - 🟡 Yellow
   - 🟢 Green
4. Vehicle counters update in real time.
5. Users can:
   - Switch to Manual Mode
   - Trigger Emergency Mode
   - Enable Pedestrian Crossing
   - Reset the Controller
